# Go to $APPDATA/Microsoft/Windows/Start Menu/Programs/Scoop Apps
# Example: C:\Users\[username]\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Scoop Apps
- Open Wezterm shortcut properties
- Under "Shortcut" tab, set "Shortcut key" to "CTRL+ALT+T"

Now you can press "CTRL+ALT+T" anywhere and it will open Wezterm
